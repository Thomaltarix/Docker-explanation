Il est ignore par Git via `.gitignore`, pour que tu puisses le garder localement sans le pousser.

- `step1/.env`
- `step2/backend/Dockerfile`
- `step2/migrator/Dockerfile`
- `step2/.env`
- `step2/docker-compose.yml`

- `step3/quote-service/Dockerfile`
